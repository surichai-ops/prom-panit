## 🎨 Customization

### เปลี่ยนสีธีม

แก้ไขใน `styles.css`:

```css
:root {
  --bg: #ffffff;
  --fg: #1a2332;
  --accent: #0d1699;        /* สีหลัก */
  --accent-light: #3b4fd8;  /* สีรอง */
  --sky: #6b8fff;           /* สีสว่าง */
  --muted: #64748b;         /* สีข้อความรอง */
}
```

### เปลี่ยนฟอนต์

แก้ไขใน `styles.css`:

```css
body {
  font-family: "Noto Sans Thai", sans-serif; /* เปลี่ยนตรงนี้ */
}
```
